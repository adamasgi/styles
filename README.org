** Basic format
A color theme css files is just a collection of variables in the :root psuedo-class.
I namespaced them with 'ctâ€™ to hopefully avoid any collisions.
An example file is included [[file:/color-themes/example.css][here]].
#+BEGIN_SRC css
:root {
	--ct-fg: #ffdec2;
	--ct-bg: #100c08;
	--ct-inv-fg: #001b32;
	--ct-alt-bg: #292623;
	--ct-green: #00a693;
	--ct-pink: #e85395;
	--ct-blue: #00aaee;
	--ct-purple: #997a8d;
}
#+END_SRC
